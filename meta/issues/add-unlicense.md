# Add an explicit project license

## Summary

Resolve GitHub issue #3 by making Headwater's open-source license explicit.

## Requirements

- License the repository under the Unlicense.
- Identify the license in package metadata and the README.
- Link the repository license from the README.

## Acceptance Criteria

- A canonical `LICENSE` file contains the Unlicense text.
- Package metadata declares `Unlicense`.
- The README has a clear licensing section.

## Notes

- GitHub issue: https://github.com/lambadalambda/headwater/issues/3
